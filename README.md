# 80nenn

四柱推命鑑定ツールの開発用リポジトリです。

## 80年年表ジェネレーター

`index.html` をブラウザで開くと、0歳の西暦を入力して80年間分の「西暦」「年齢」「干支」を一覧表示し、PNG形式でダウンロードできるツールを利用できます。

### 使い方
1. 0歳の西暦（生まれ年）を入力します。
2. 「一覧を作成」をクリックすると表が生成されます。
3. 「PNGとしてダウンロード」をクリックすると、表示中の表をPNG画像として保存できます。

### 表示が更新されないときは
ブラウザが以前の `styles.css` を保持している場合があります。最新のレイアウトが表示されない場合は、以下をお試しください。
- Windows: `Ctrl + F5`
- macOS: `Shift + Command + R`
- もしくはブラウザのキャッシュを削除してから再度 `index.html` を開いてください。
