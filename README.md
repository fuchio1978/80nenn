# 80nenn

四柱推命鑑定ツールの開発用リポジトリです。

## 80年年表ジェネレーター

`index.html` をブラウザで開くと、0歳の西暦を入力して80年間分の「西暦」「年齢」「干支」を一覧表示し、PNG形式でダウンロードできるツールを利用できます。

### 使い方
1. 0歳の西暦（生まれ年）を入力します。
2. 「一覧を作成」をクリックすると表が生成されます。
3. 「PNGとしてダウンロード」をクリックすると、表示中の表をPNG画像として保存できます。

### 表示が更新されないときは
以下のいずれかが原因で、古い緑色のレイアウトが残っている可能性があります。

1. **まだ `main` ブランチへマージできていない**  
   Pull Request でコンフリクトを解消しただけでは `main` に反映されません。`Merge pull request` → `Confirm merge` まで完了しているかを確認し、完了していなければマージを行ってください。
2. **ダウンロードしたフォルダに旧ファイルが残っている**  
   同じフォルダ名に解凍すると OS が既存ファイルを残すことがあります。新しいフォルダ名で展開するか、既存フォルダを削除してから再度 ZIP を解凍してください。
3. **ブラウザに古いスタイルがキャッシュされている**  
   Windows では `Ctrl + F5`、macOS では `Shift + Command + R` を押してハードリロードし、キャッシュをクリアした状態で読み込んでください。
